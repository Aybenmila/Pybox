# PYBOX

A 2d physic based sandbox backbone made with python and pygame. 

This backbone provides 5 different particle types so far:

* Nuclear
* Powder
* Solid
* Water
* Gas

### TEST REACTIONS

1. Electron + Proton = Neutron
